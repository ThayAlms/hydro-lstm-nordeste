# hydro-lstm-nordeste
Predicting water volume levels of the Armando Ribeiro Gonçalves Reservoir (RN, Brazil) using Bidirectional LSTM and Isolation Forest for anomaly detection. Historical data from 1987–2023 combining ANA telemetry, ERA5 reanalysis, and the ONI index (El Niño/La Niña). Test MAE of 2.68%.
